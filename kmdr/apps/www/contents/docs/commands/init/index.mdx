## Usage

```bash
npx k0nfig@latest init
```

This `command.config.json` file stores the state and context of your project and is stored at the root of the project directory. It is mandatory to run init before using the `run` command.

By default running init command will:

1. Ask the user for which directory will they want to add all files copied from the configuration to.
2. Ask the user which package mangaer do they want to use.
